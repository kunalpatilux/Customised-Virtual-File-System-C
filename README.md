# Customised Virtual File System (CVFS)

A Customised Virtual File System implemented in C that simulates the behavior of the Linux File System.  
The system implements basic file system operations such as create, read, write, delete, and list files using self-defined data structures.

## Features

- Create files
- Write data into files
- Read file content
- Delete files
- List files
- Custom shell interface
- Simulated Linux file system environment

## Technologies Used

- C Programming
- Data Structures
- System Programming
- File System Concepts

## Data Structures Implemented

- Incore Inode Table
- File Table
- User File Descriptor Table (UFDT)

## Learning Outcomes

- Deep understanding of Linux file system internals
- Implementation of OS data structures
- Shell command interpreter design
- System programming concepts in C
